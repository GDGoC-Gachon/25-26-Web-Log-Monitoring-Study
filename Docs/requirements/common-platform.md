# 공통 실행 기반 요구사항

대상 코드: `src/app.ts`, `src/config.ts`, `src/utils/elastic-query.client.ts`, `src/utils/elastic-user.client.ts`

## 책임 범위

- TypeScript / Node.js 실행 기반을 제공한다.
- 모든 탐지 잡에서 공유하는 설정, 인증 정보, Elasticsearch 로그 조회, 로그 파싱, 로깅, 주기 실행 구조를 관리한다.
- 2차 개발 범위의 방화벽 차단 자동화와 공통 품질 요구사항을 추적한다.
