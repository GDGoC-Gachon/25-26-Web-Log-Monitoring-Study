# 무차별 대입 공격 탐지 요구사항

대상 코드: `src/jobs/brute-force.job/job.ts`

## 책임 범위

- 로그인 또는 인증 경로에서 발생하는 반복 실패 요청을 탐지한다.
- 동일 `client_ip`와 `domain` 조합을 기준으로 악성 IP 여부를 판단한다.
- 설정된 예외 IP는 탐지 대상에서 제외한다.

## 공통 의존 요구사항

- REQ-002: 탐지 시간 범위, 실패 횟수 기준, 예외 IP 목록은 config에서 관리한다.
- REQ-004: 최근 n분 로그를 Elasticsearch ESQL API로 조회한다.
- REQ-005: `timestamp`, `client_ip`, `domain`, `path`, `protocol_status`를 내부 분석 구조로 파싱한다.
- REQ-018: 알림 본문에 탐지 유형, 대상 IP/domain, 발생 횟수, 탐지 기준, 탐지 사유가 전달될 수 있어야 한다.
- REQ-026: 탐지 결과 생성과 실패 상황을 로그로 확인할 수 있어야 한다.
- REQ-028: 정상 탐지, 미탐지, 예외 IP 케이스를 단위 테스트로 검증한다.
