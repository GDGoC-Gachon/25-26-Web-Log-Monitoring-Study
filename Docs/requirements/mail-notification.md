# 메일 알림 요구사항

대상 코드: `src/jobs/mail-notification.job/job.ts`

## 책임 범위

- 탐지 결과를 이메일 알림으로 전달한다.
- Elasticsearch 사용자 정보를 기준으로 수신 대상을 산정한다.
- 관리자와 서비스 사용자의 알림 범위를 구분한다.
- 이메일 누락 사용자와 중복 알림을 운영상 안전하게 처리한다.

## 탐지 잡 의존 관계

- `brute-force.job`: 악성 IP, domain, 실패 횟수, 탐지 기준, 탐지 사유를 알림 대상으로 전달한다.
- `DDos.job`: 악성 IP, domain, 요청 횟수, 탐지 기준, 탐지 사유를 알림 대상으로 전달한다.
- `web-error.job`: domain, 4xx 발생 횟수, 오류 비율, 탐지 기준, 탐지 사유를 알림 대상으로 전달한다.
- `server-error.job`: domain, 5xx 발생 횟수, 오류 비율, 탐지 기준, 탐지 사유를 알림 대상으로 전달한다.

## 공통 의존 요구사항

- REQ-003: Elasticsearch와 Resend API 인증 정보는 `.env`에서 관리한다.
- REQ-014: 사용자 조회 결과는 알림 대상 선정의 입력이다.
- REQ-026: 사용자 조회 실패, 이메일 누락, 메일 발송 실패, 발송 성공을 로그로 확인할 수 있어야 한다.
